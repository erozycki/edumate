First steps

Try reading the overview to get a basic understanding of what Edumate does, or jump right in by creating an account!

Once you have an account, add a deck and review any cards that are due. Each card consists of a prompt and an answer. The prompt is presented first - try to answer it as best as you can, and then press any key to reveal the answer. Finally, indicate your performance; press "Wrong" if you got the item wrong, "Right" if you got it right, and "Trivial" if the item was so easy that you don't think you'll need to review it very often.
